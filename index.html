Private URL Shortener

Static App • IndieWeb • Personal Cloud

Do This:

  1. Create dedicated domain on an Apache server that supports .htaccess
  2. ssh to root directory there
  3. curl http://s.gonze.com/starter.txt > .htaccess
  4. When you want to shorten a URL, edit .htaccess. Replace any URL_TO_SHORTEN with the long URL.

Why is this a good method?

  • Your redirects should last for years with no maintenance or security problems. The data is in a configuration file, so there are no complex components, like databases or scripts, to break down.
  • Your links are hard to spider. The short URLs aren't predictable serial numbers, they are random strings.
