shurl

URL shortener

This app is special in that it uses the Apache mod_rewrite file to store rewrites. There is no database, only a static file. There are no complex components to break down or be maintained.

Your rewrites should last for a long long time with little maintenance or security problems.

The system works using a client-side console app which modifies .htaccess on your server. The first time you run it,

Do This:

Create dedicated domain on an Apache server that supports .htaccess
ssh to root directory there
curl http://s.gonze.com/starter.txt > .htaccess
When you want to shorten a URL, edit .htaccess. Replace any URL_TO_SHORTEN with the long URL.
